# AIDLC 人をだめにする案

## 背景
AIDLC ハッカソンにて、人×AIの力を活用して『人をだめにするサービス』作成を考慮する大会があります。
世の中の情勢や、人が一体何を求めどんなサービスを作成することでダメにすることができるか考慮しています。

## ゴール
考慮した案を以下のフェーズまで考慮すること
https://github.com/awslabs/aidlc-workflows/blob/main/aidlc-rules/aws-aidlc-rules/core-workflow.md

## 依存に対する考慮点
サービス案を出すにあたり、依存観点について洗い出します。

### 人は日常的に何に依存して過ごしているのか？
- 行動・習慣系
    - スマートフォン
    - SNS(X, Instagram)
    - 動画媒体(Netflix, Youtube, Amazon Prime)
    - ショッピング
- 物質系
    - カフェイン（コーヒー、エナジードリンク）— 摂取しないと頭痛や倦怠感が出やすい
    - アルコール — 習慣化しやすく、耐性がつきやすい
    - 糖分・甘いもの — 血糖値の変動で「また食べたい」感覚が生まれる
- 心理・関係系
    - 承認欲求（「いいね」を求める行動）
    - 特定の人間関係への依存（共依存）
    - 過食・制限食など食にまつわる行動パターン
- 環境・感覚系
    - 音楽・ノイズ（無音が落ち着かない）
    - 特定の場所・空間（カフェでないと作業できない）

### なぜ人間は依存してしまうのか
- ストレス
    - ストレス解消きっかけでやめられなくなった
- 退屈・刺激不足
    - 空虚な状態を埋めるための行動。スマホを開く多くのケースはこれに近い。
- アイデンティティ・習慣の固定化
    - 自分はコーヒーを飲む人間だ」という自己イメージが行動を固定する。やめる理由がないまま継続される。
-  社会的同調圧力
    - 飲み会のアルコール、職場のコーヒー文化など、集団に属するための行動が習慣化するケース。
-  生理的快楽・報酬回路
    - ストレスとは無関係に、単純に「気持ちいいから」。糖分やカフェインは脳の報酬系を直接刺激する。
- 時間構造の代替
    - 仕事が終わった後にNetflixを見るなど、「次に何をするか」の意思決定コストを減らすルーティン化。
- 不確実性の回避
    - 予測可能な刺激（同じ作品の繰り返し視聴、同じ店に通うなど）で安心感を得る。新奇なものより既知のものを選ぶ。

上記のように複数ある依存の原因の中で、スマホは退屈・報酬回路・習慣化を同時に刺激できる唯一のデバイスであるため、依存性と替えがきかない存在である

### 普段の生活で必ず必要なこと
以下は日常で過ごすうえで必ず必要な項目だが、外に出なくてもスマホなどで対応できてしまっている
- 睡眠
- 食事
- 買い物

### 24時間の内訳（全年代平均）

| カテゴリ | 時間 | 割合 | データ種別 |
|---|---|---|---|
| 睡眠 | 7h 41m | 32% | 実測・信頼度高 |
| スクリーンタイム（純粋） | 5h 30m | 23% | 実測・信頼度高 |
| 仕事・学校 | 6h 04m | 25% | 調査値・参考 |
| 食事・身支度 | 2h 00m | 8% | 調査値・参考 |

> 出典：総務省統計局「社会生活基本調査（令和3年）」、総務省情報通信政策研究所「情報通信メディアの利用時間と情報行動に関する調査（令和6年度）」

### スクリーンタイム 5h30m の内訳

| 用途 | 時間/日 | 割合 |
|---|---|---|
| SNS・メッセージ | 90分 | 27% |
| 動画視聴 | 75分 | 23% |
| ウェブ閲覧 | 60分 | 18% |
| 仕事・学習 | 43分 | 13% |
| ゲーム | 40分 | 12% |
| その他 | 22分 | 7% |

> 出典：総務省情報通信政策研究所（令和6年度）、DataReportal 2024、Mikke調査

### データから読み取れる示唆

- **1日の約23%がスクリーンタイム**（5時間30分）であり、仕事・学校時間（25%）とほぼ同等
- スクリーンタイムのうち **SNS・動画・ウェブで約68%** を占め、娯楽・受動的消費が中心
- スクリーンタイムは仕事・家事中のながら利用を含むため、**他の時間帯と重複して存在する**
- 睡眠・食事・仕事を除いた「自由に使える時間」はごくわずかであり、その時間の大半をスマホが占有している

→ スマホ・スクリーンタイムが「空き時間の受け皿」として機能しており、代替となるサービス設計で依存を誘導しやすい構造がある

## 現代人が抱える悩みと意思決定負荷
 
依存の構造を踏まえ、現代人が日常的に抱える「悩みや負荷」を整理する。これがサービスが解決すべき課題の出発点となる。
 
### 現代人の主な悩み領域
 
| 領域 | 代表的な悩み | 特徴 |
|---|---|---|
| 決断疲れ | 選択肢が多すぎて選べない・選ぶこと自体が疲れる | 毎日繰り返し発生する |
| 慢性的ストレス・こころの不調 | 健康リスクとして「ストレス」を挙げる割合が20年で3倍に増加（厚労省2024年白書） | 30〜40代が特に深刻 |
| 疲労の蓄積 | 「疲れている人」が推計7,162万人（計測以来のピーク）、睡眠5時間未満が増加 | 睡眠不足と悪循環 |
| 失敗恐怖・行動抑制 | SNSで他者の失敗が可視化され、「安全な選択」しかできなくなる | 特に若い世代に顕著 |
| きずな依存・孤独感 | SNSの人間関係は負担だが、抜け出せない。依存者の52%が「SNS上の人間関係に負担を感じている」 | 日本人特有の傾向 |
 
### 意思決定負荷の深刻さ
 
人は1日に約3.5万回の意思決定をしていると言われている。現代では選択肢の爆発的な増加により、この負荷がさらに増大している。
 
**決断疲れ（Decision Fatigue）** とは、意思決定を繰り返すことで脳が疲弊し、後半になるほど「どうでもいい」と投げやりな選択をしてしまう現象。Netflixで何を見るか決められず結局閉じる、というのがその典型例。
 
### 意思決定が発生する主な場面
 
意思決定の場面は大きく4領域に分けられるが、「依存しやすいサービス」を設計するうえでは、**繰り返し発生する日常的な意思決定**が最も有効なターゲットになる。
 
| 領域 | 代表的な場面 | 依存しやすさ |
|---|---|---|
| **日常生活** | 献立・食事・買い物・服装・余暇の使い方 | ◎ 毎日繰り返される |
| 仕事・業務 | メール返信・タスク優先順位・会議での発言 | ○ 毎日発生するが職種依存 |
| 消費・購買 | EC商品選択・サブスクプラン・旅行先 | ○ 頻度は高いが散発的 |
| キャリア・人生 | 転職・住まい・学習テーマ | △ 重大だが頻度が低い |
 
→ **日常生活領域**は毎日必ず発生し、失敗しても取り返しがつく軽さがある。そのため「AIに任せる」習慣が最もつきやすく、依存ループが形成されやすい。
 
 
## サービス方向性
 
### コンセプト
 
**「考えなくていい」が癖になる AI お任せ体験**
 
30〜40代が毎日繰り返す日常生活の意思決定を、AIが丸ごと引き受けることで「考える消耗」から解放する。一度任せると楽になることを体感し、委ねる範囲が自然と広がっていく依存ループを設計する。
 
### ターゲット
 
- **年代**：30〜40代（社会人・子育て世代を含む働き盛り）
- **ペイン**：仕事・家事・育児で毎日膨大な判断をしており、帰宅後や休日にも「次に何をするか」を考えることに疲れている
- **インサイト**：「楽をしたい」より「考えたくない」が本音。選択肢があること自体がストレス
### 対象とする意思決定（日常生活領域）
 
- 今日の献立・食事
- 今週の買い物リスト
- 今夜・休日の過ごし方
- 今日着る服装
- その他、毎日繰り返される小さな選択
### 依存が生まれるメカニズム
 
1. **入口**：「今日の夕飯、AIに決めてもらう」という小さな体験
2. **快感**：考えなくていい解放感・選んでもらった安心感
3. **習慣化**：繰り返すうちに「自分で考えるのが面倒」になる
4. **拡張**：委ねる範囲が献立 → 買い物 → 休日 → 服装と広がっていく
5. **依存完成**：AIなしで日常の選択ができなくなる
